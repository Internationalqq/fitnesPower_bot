ИНСТРУКЦИЯ ПО ЗАПУСКУ БОТА:

1. Установи зависимости:
   pip install -r requirements.txt

2. Создай файл .env в корне проекта:
   BOT_TOKEN=твой_токен_от_BotFather

3. Запусти бота:
   python bot.py

4. Добавь бота в группу и дай ему права администратора (для отправки сообщений)

5. В группе используй команды:
   /pushups 80 - добавить отжимания
   /abs 80 - добавить упражнения на пресс
   /stats - статистика за сегодня
   /my_stats - твоя статистика
   /leaderboard - таблица лидеров

6. В личке с ботом:
   Просто напиши что съел: "овсянка 200г, банан 1шт"
   Или используй команды:
   /today - калории за сегодня
   /week - статистика за неделю
   /set_limit 2000 - установить норму калорий

Бот автоматически будет отправлять мотивирующие факты и советы в 9:00 и 18:00 в группы, где он активен.
